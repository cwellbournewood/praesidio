# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), v2.1.

In short: be excellent to each other. Harassment, discrimination, and
personal attacks are not tolerated. Report concerns by opening a private
report at <https://github.com/cwellbournewood/section/security/advisories/new>
or by emailing the project maintainer listed in `CONTRIBUTING.md`.
