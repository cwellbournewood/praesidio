Praesidio {{ .Chart.AppVersion }} has been installed as release "{{ .Release.Name }}" in namespace "{{ .Release.Namespace }}".

1. Verify the pods are running:

   kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/instance={{ .Release.Name }}

2. Run the bundled smoke test:

   helm test {{ .Release.Name }} -n {{ .Release.Namespace }}

3. Port-forward to the gateway and the admin UI locally:

   # Gateway (OpenAI-compatible API)
   kubectl -n {{ .Release.Namespace }} port-forward svc/{{ include "praesidio.componentName" (dict "ctx" . "component" "gateway") }} 8080:{{ .Values.service.gateway.port }}

   # Admin UI
   kubectl -n {{ .Release.Namespace }} port-forward svc/{{ include "praesidio.componentName" (dict "ctx" . "component" "ui") }} 3000:{{ .Values.service.ui.port }}

4. Send a test request:

   export OPENAI_BASE_URL=http://localhost:8080/v1
   export OPENAI_API_KEY=praesidio-demo-key
   curl $OPENAI_BASE_URL/chat/completions \
     -H "Authorization: Bearer $OPENAI_API_KEY" \
     -H "Content-Type: application/json" \
     -d '{"model":"gpt-4o-mini","messages":[{"role":"user","content":"hello"}]}'

5. Grafana dashboards live in `deploy/grafana/dashboards/`:
   - `praesidio-overview.json`  — throughput, decisions, detector latencies
   - `praesidio-compliance.json` — per-tenant audit volume, FP rate, retention
