## Known caveats

- **Agent broker** (signed capability tokens, sandboxed tool execution)
  remains architected; the tool-call allowlist landed as the runtime
  half. See `docs/architecture/07-agent-governance.md` for the gap.
- **Qdrant connector** ships an in-memory ACL backend by default;
  production deployments MUST configure a real ACL store (documented
  in the module docstring).
- **Helm chart locally**: `helm lint` is run in CI
  (`.github/workflows/helm-upgrade.yml`) rather than the local sign-off
  workflow.
